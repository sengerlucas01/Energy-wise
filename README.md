# ⚡ EnergyWise

Sistema web inteligente desenvolvido para auxiliar vendedores de energia solar na apresentação de propostas comerciais e cálculo de payback para clientes residenciais e empresariais.

O sistema permite inserir os dados do cliente e gerar uma análise detalhada do retorno financeiro do investimento em energia solar, ajudando no processo de convencimento comercial de forma visual, prática e profissional.

---

# Objetivo

Este projeto tem como objetivo praticar desenvolvimento full stack com uma arquitetura moderna, organizada e adequada para portfólio.

A proposta do sistema é facilitar o processo de venda de sistemas fotovoltaicos, permitindo que vendedores demonstrem de forma clara:

- Economia mensal do cliente
- Tempo estimado de retorno do investimento (payback)
- Comparação de gastos atuais vs economia futura
- Simulação personalizada para residências e empresas
- Geração de relatórios em PDF para apresentação comercial

---

# Funcionalidades Do Projeto

- Landing page moderna e responsiva
- Simulação de investimento em energia solar
- Cálculo automático de payback
- Cadastro de dados do cliente
- Simulação para residência e empresa
- Exibição detalhada de economia mensal
- Relatório financeiro visual
- Geração de PDF para envio ao cliente
- Interface intuitiva para vendedores
- Deploy online via Netlify
